# Image Processing and Feature Detection in Python

Ini adalah proyek yang mengeksplorasi teknik-teknik fundamental dalam Image Processing dan Feature Detection menggunakan Python, OpenCV, dan NumPy.

## Fitur Utama

- **Analisis Komparatif Filter:** Menganalisis secara komparatif teknik image processing di Domain Spasial (filter konvolusi Sobel, Canny, LoG) dan Domain Frekuensi (filter berbasis FFT/Gaussian).
- **Analisis Filter Trade-off:** Menganalisis trade-off berbagai filter, termasuk mendemonstrasikan keunggulan Median Blur dibanding Gaussian Blur untuk mereduksi salt-and-pepper noise.
- **Implementasi Harris Corner Detector:** Mengimplementasikan proses deteksi fitur algoritma Harris Corner Detector.
- **Custom Non-Maximum Suppression (NMS):** Membuat NMS kustom untuk ekstraksi fitur corner yang lebih akurat.
